# worker-hub

## Commit Message Convention

| Type  |                      내용                      |
| :---: | :--------------------------------------------: |
| feat  |                새로운 기능 추가                |
|  fix  |                   버그 수정                    |
| docs  |                   문서 관련                    |
| style | 스타일 변경 (포매팅 수정, 세미 콜론 추가, ...) |
| test  |                테스트 관련 코드                |
| chore |                   설정 변경                    |

## Code Convention

- Component 파일명은 pascal case 사용
- ESLint 기반 스타일 준수
- Prettier를 적용하여 코드 컨벤션 유지
